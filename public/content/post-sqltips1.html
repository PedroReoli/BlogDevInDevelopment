Comandos SQL Básicos: Introdução para Iniciantes

Aprenda os fundamentos do SQL com exemplos simples para começar a manipular dados em um banco de dados.

Introdução ao SQL

SQL (Structured Query Language) é a linguagem utilizada para interagir com bancos de dados relacionais. Neste artigo, vamos explorar os comandos SQL básicos para que você possa começar a trabalhar com dados em tabelas. Vamos abordar a criação de tabelas, manipulação de dados e a execução de consultas simples.

Exemplo 1: CREATE TABLE

O comando CREATE TABLE é utilizado para criar novas tabelas no banco de dados. Ao criar uma tabela, você define os nomes das colunas e seus tipos de dados.

Comando Resultado Esperado 
CREATE TABLE clientes (
  id_cliente INT PRIMARY KEY,
  nome NVARCHAR(100),
  idade INT,
  cidade NVARCHAR(100)
);
            
Tabela 'clientes' criada com sucesso.

Exemplo 2: ALTER TABLE

O comando ALTER TABLE é usado para modificar a estrutura de uma tabela existente. Com ele, é possível adicionar, remover ou modificar colunas em uma tabela já criada.

Comando Resultado Esperado 
ALTER TABLE clientes
ADD telefone NVARCHAR(15);
            
Coluna 'telefone' adicionada à tabela 'clientes'.

ALTER TABLE para renomear coluna

O comando ALTER TABLE pode ser usado para renomear colunas de uma tabela.

Comando Resultado Esperado 
ALTER TABLE clientes
RENAME COLUMN telefone TO celular;
          
| nome        | celular |
|-------------|---------|
| João Silva  | NULL    |
| Ana Costa   | NULL    |

Exemplo 3: SELECT

O comando SELECT é utilizado para consultar dados de uma ou mais tabelas. Ele é o comando principal para recuperar informações de um banco de dados.

Comando Resultado Esperado 
SELECT nome, idade FROM clientes;
            
| nome       | idade |
|------------|-------|
| João Silva | 35    |
| Maria Lima | 42    |

Exemplo 4: INSERT INTO

O comando INSERT INTO é utilizado para adicionar novos registros em uma tabela.

Comando Resultado Esperado 
INSERT INTO clientes (nome, idade, cidade)
VALUES ('Carlos Souza', 28, 'São Paulo');
            
| nome        | idade | cidade    |
|-------------|-------|-----------|
| Carlos Souza| 28    | São Paulo |

Exemplo 5: UPDATE

O comando UPDATE é utilizado para modificar registros existentes em uma tabela.

Comando Resultado Esperado 
UPDATE clientes
SET cidade = 'Rio de Janeiro'
WHERE id_cliente = 5;
            
| nome        | idade | cidade          |
|-------------|-------|-----------------|
| João Silva  | 35    | Rio de Janeiro  |

Exemplo 6: DELETE

O comando DELETE é utilizado para remover registros de uma tabela.

Comando Resultado Esperado 
DELETE FROM clientes WHERE id_cliente = 5;
            
| nome        | idade | cidade       |
|-------------|-------|--------------|
| João Silva  | 35    | São Paulo    |
| Maria Lima  | 40    | Rio de Janeiro|

Conclusão

Com esses comandos básicos de SQL, você agora pode começar a criar, consultar, atualizar e excluir dados em bancos de dados relacionais. Esses são os fundamentos para começar a trabalhar com SQL e bancos de dados. Pratique e continue explorando comandos mais avançados para aprofundar seus conhecimentos.
