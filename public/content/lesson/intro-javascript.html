Introdução ao JavaScript

Uma jornada completa pelo mundo do JavaScript.

Criação e Primeiros Passos

O JavaScript foi criado por Brendan Eich em 1995, enquanto ele trabalhava na Netscape Communications Corporation. Inicialmente, a linguagem foi chamada de "Mocha", depois "LiveScript", e finalmente recebeu o nome de "JavaScript" como estratégia de marketing, aproveitando a popularidade do Java.

Apesar do nome, JavaScript e Java são linguagens completamente diferentes. Java é uma linguagem compilada e orientada a objetos, enquanto o JavaScript é interpretado e voltado para o desenvolvimento web interativo.

Papel da Netscape e da Microsoft

A Netscape introduziu o JavaScript para permitir interatividade em sites, mas a Microsoft logo criou sua própria versão chamada JScript. Isso resultou em inconsistências, dificultando o desenvolvimento web até o surgimento do padrão ECMAScript pela ECMA International, em 1997.

O que é ECMAScript?

O ECMAScript é a padronização do JavaScript. A cada versão, novos recursos são introduzidos. As atualizações mais importantes foram:

  • ES3 (1999): Primeira versão consolidada do JavaScript.
  • ES5 (2009): Introduziu recursos como JSON e métodos funcionais.
  • ES6 (2015): Adicionou classes, arrow functions, let/const, e módulos.

Sintaxe Básica e Declaração de Variáveis

O JavaScript permite declarar variáveis de diferentes maneiras: 


// Usando let
let nome = "João";

// Usando const (imutável)
const idade = 25;

// Usando var (não recomendado)
var cidade = "São Paulo";

Use let para variáveis mutáveis e const para valores constantes. Evite var, pois possui um comportamento menos previsível.

Funções no JavaScript

As funções são blocos reutilizáveis de código. Veja um exemplo: 


// Função tradicional
function saudacao(nome) {
  return `Olá, ${nome}!`;
}

// Arrow function
const saudacaoArrow = (nome) => `Olá, ${nome}!`;
console.log(saudacao("Pedro"));

DOM e Manipulação de Eventos

O JavaScript permite manipular elementos HTML diretamente. Por exemplo: 


// Alterando texto de um elemento
document.getElementById("titulo").textContent = "Novo Título";

// Adicionando um evento de clique
document.getElementById("botao").addEventListener("click", () => {
  alert("Botão clicado!");
});
